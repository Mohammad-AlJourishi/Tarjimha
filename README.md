# Tarjimha

![Tarjimha Homepage](homepage.png)

Tarjimha is a real-time translation web application that allows users to speak or type a message, translate it into another language, hear the translated result aloud, and view ASL fingerspelling support.

## Features

- Real-time voice recognition
- Text translation
- Text-to-speech output
- ASL fingerspelling display
- Conversation mode
- Transcript history
- Export transcript
- Presentation mode

## Supported Languages

- Arabic
- English
- French

## Technologies Used

- HTML
- CSS
- JavaScript
- Web Speech API
- Google Translate endpoint
- Wikimedia Commons ASL hand images

## How to Use

1. Choose the source language.
2. Choose the target language.
3. Click the microphone button.
4. Speak your message.
5. Tarjimha translates it, speaks it aloud, and shows ASL fingerspelling.

You can also type a message and click Translate.

## Browser Requirements

Use Google Chrome or Microsoft Edge.

Speech recognition requires HTTPS or localhost. GitHub Pages provides HTTPS.
